# Мир Держав — сайт Военно‑Политического Minecraft-сервера

Статический многостраничный сайт без сборщика и внешних библиотек.

## Страницы

- `index.html` — компактная главная
- `about.html` — информация о сервере и интерактивная карта-заглушка
- `news.html` — новости проекта со скриншотами
- `mods.html` — список модов и блок загрузки мод-пака

## Запуск

Откройте `index.html` в браузере либо разместите папку на любом статическом хостинге.

Для локальной проверки через HTTP:

```bash
python -m http.server 8080
```

## Что нужно заменить позже

### Ссылка на мод-пак

В `mods.html` найдите блок `download-placeholder`. Когда появится ссылка, замените кнопку на:

```html
<a class="primary-action" href="ССЫЛКА" target="_blank" rel="noopener">Скачать мод-пак</a>
```

### Подключение живой карты

Интерактивная демонстрация находится в `about.html` внутри элемента `data-map`.
Её можно заменить:

- iframe BlueMap;
- iframe Dynmap;
- собственным интерфейсом, получающим данные через API/WebSocket.

### Новые новости

Добавляйте новый `<article class="news-article">` в `news.html`.
Скриншоты хранятся в `assets/images`.
