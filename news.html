01 17.07.2026

Запущен новый сайт проектаThe project website is live

У Военно‑Политического сервера появился официальный сайт. Теперь новости разработки, информация о мире, актуальный список модов и будущая интерактивная карта собраны в одном месте.

Сайт будет развиваться вместе с проектом. После подключения серверной карты здесь появятся государства, города, границы и другие данные из игрового мира. Также на сайте будет опубликована ссылка для загрузки готового мод-пака.

Все важные объявления и даты тестирования по-прежнему будут дублироваться в Discord.

The Military-Political server now has an official website. Development news, world information, the current mod list and the future interactive map are now collected in one place.

The website will develop together with the project. Once the server map is connected, states, cities, borders and other in-game data will appear here. A download link for the complete modpack will also be published.

All important announcements and test dates will continue to be posted in Discord.

02 16.07.2026

Подготовка сервераServer preparation

На сегодняшний день большая часть работы над сервером завершена. Уже добавлены все необходимые клиентские моды, разработаны серверные моды, а работа над системой безопасности подходит к завершению.

В скором времени мы планируем провести открытый двухдневный бета-тест сервера, во время которого любой желающий сможет присоединиться к игре.

Цель короткого запуска — найти возможные баги, собрать предложения и пожелания игроков, провести нагрузочное тестирование и выявить оставшиеся недочеты.

Во время теста можно будет свободно исследовать сервер, строить, ломать, создавать государства и выполнять любые другие действия. После завершения тестирования мы внесем необходимые исправления и подготовим сервер к полноценному открытию.

Most of the work on the server has now been completed. All required client-side mods have been added, server-side mods have been developed, and the security system is nearing completion.

We plan to hold a public two-day beta test soon, during which anyone will be able to join the server.

The purpose of this short launch is to identify bugs, gather player suggestions, conduct stress testing and uncover any remaining issues.

During the test, players will be free to explore, build, destroy, create states and perform any other in-game activities. Afterward, we will make the necessary fixes and prepare the server for the official launch.

03 19.06.2026

Начало развития положеноDevelopment has begun

Мы заложили первый фундамент сервера. Уже арендовано мощное серверное оборудование, проведены технические тесты, добавлены первые моды и плагины.

Проект постепенно переходит из стадии заморозки в полноценную рабочую платформу.

Следить за процессом разработки и обновлениями можно в Discord. Предложения по новым модам также принимаются в отдельном канале сообщества.

We have laid the first foundation of the server. Powerful server hardware has been rented, technical tests have been completed, and the first mods and plugins have been added.

The project is gradually moving from a frozen state into a fully functional platform.

You can follow the development process and updates in Discord. Suggestions for additional mods are also accepted in a dedicated community channel.

04 14.06.2026

Запуск экономического ботаEconomy bot launched

Мы запустили Discord-бота, который отвечает за экономику сервера.

В нём уже доступны личные балансы, бюджеты стран, переводы между игроками и государствами, а также рейтинг стран.

Бот активен и уже используется участниками проекта.

We have launched a Discord bot responsible for the server economy.

It already supports personal balances, country budgets, transfers between players and states, and country rankings.

The bot is active and already used by project members.

05 ИЮНЬ 2026JUNE 2026

Возрождение проектаProject revival

После долгого перерыва мы приняли решение вернуть к жизни Военно‑Политический сервер Minecraft в модернизированном формате.

Первым этапом стало создание полноценной экономической системы проекта — Discord-бота, регулирующего оборот денежных средств, ресурсов и финансовых операций.

После этого началась разработка необходимых серверных решений, интеграция модов и перенос проекта на производительную инфраструктуру для стабильного доступа игроков.

After a long hiatus, we decided to bring the Military-Political Minecraft server back to life in a modernized format.

The first stage was the creation of a complete project economy — a Discord bot regulating money, resources and financial transactions.

This was followed by the development of required server solutions, mod integration and migration to high-performance infrastructure for stable player access.
